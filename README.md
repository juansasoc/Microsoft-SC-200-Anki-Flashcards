## üìñ How to Import into Anki
1. Download the `.tsv` file you want.  
2. In Anki: **File ‚Üí Import**.  
3. Select the file.  
4. Choose **Tab-separated**, with **Field 1 = Front** and **Field 2 = Back**.  
5. Import ‚Üí start studying üöÄ
