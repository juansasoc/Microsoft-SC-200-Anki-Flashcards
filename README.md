

**SC-200 Anki Flashcards**

This repository provides a clean, purely technical SC-200 study deck designed to help learners build deep understanding of Microsoft 365 Defender, Defender for Cloud, Microsoft Sentinel, and KQL—without any exam-specific language or memorization cues.

It is ideal for anyone who wants to strengthen real-world security operations knowledge while preparing for the SC-200 exam or improving daily SOC workflow skills.

---

**What’s Included**

Each domain and topic area includes two file formats:

**.md (Markdown)**

* Easy to read directly on GitHub
* Great for browsing or reviewing concepts without Anki

**.tsv (Tab-Separated Values)**

* Fully compatible with Anki
* Import as:

  * Note Type: Basic (front/back)
  * Delimiter: Tab
  * Allow HTML: Enabled

---

**Deck Structure**

The repo is organized into six technical areas:

1. Domain 1: Microsoft 365 Defender
2. Domain 2: Microsoft Defender for Cloud
3. Domain 3: Microsoft Sentinel
4. Bonus KQL: Core operators and query fundamentals
5. Scenario KQL: Detection and investigation patterns
6. KQL Mastery: Advanced query techniques and logic

Each folder contains 30 concise, high-yield Q&A flashcards focused on practical usage, detections, threat investigation, and daily SOC workflows.

---

**Using the Flashcards in Anki**

1. Open Anki
2. Go to File → Import
3. Select the .tsv file from the topic you want to add
4. Import using the recommended settings
5. Start reviewing using spaced repetition
